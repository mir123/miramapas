Corredores naturales: mapa dinámico en la web

  • OpenLayers
  • Para: Almanaque Azul
  • Mir Rodríguez Lombardo, 2022-2024

Hice este mapa dinámico para acompañar el mapa de corredores naturales de Panamáy poder ver los datos con mayor detalle.

El mapa permite visualizar en detalle los datos de conectividad que obtuvimos del modelo. Por medio de un slider se pueden cambiar los niveles de conductividad que se muestran. Los resultados del modelo tienen una resolución de 30 m, por lo que es posible acercarse y estudiar los efectos de conectividad a nivel de finca o de manzana urbana.

Todos los mapas y los datos están disponibles públicamente en el sitio web de Almanaque Azul.

Este mapa está desarrollado en OpenLayers y las capas se cargan usando la fantástica tecnología serverless que ahora tenemos disponible: Protomaps y Cloud Optimized GeoTIFF. Gracias a eso es posible poner los datos en un hosting web barato, sin necesidad de un servidor geográfico como GeoServer o ArcGIS Server.
